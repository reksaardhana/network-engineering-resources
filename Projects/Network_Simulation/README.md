# Proyek Simulasi Jaringan

Berikut adalah beberapa proyek simulasi jaringan yang dapat Anda coba.

## 1. Simulasi Jaringan Perkantoran
- **Deskripsi**: Simulasi jaringan untuk kantor dengan beberapa VLAN.
- **Perangkat**: 1 Router, 2 Switch, 10 PC.
- **Langkah**:
  1. Buat topologi dengan perangkat.
  2. Konfigurasi VLAN dan inter-VLAN routing.

## 2. Simulasi Jaringan Rumah
- **Deskripsi**: Simulasi jaringan rumah dengan perangkat IoT.
- **Perangkat**: 1 Router, 1 Switch, 5 PC, 3 IoT devices.
- **Langkah**:
  1. Buat topologi dengan perangkat.
  2. Konfigurasi DHCP dan NAT.
